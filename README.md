# Platformer_Project_Game_Development
The platformer project made during the Game Development subject - Second year of videogame development

## Team
- Fernando Freixinet
- Marta Llurba

## Controller:
- A: Move left
- D: Move right
- SPACE: Jump
- RIGHT CLICK: attack

## Debug Keys
- F5: Load the current game state
- F6: Load theprevious state
- F9: Show colliders
- F10: God Mode
